公文归档：

对已经流转结束的发文、收文进行归档。

执行条件：

系统正常运行。

操作说明：

进入模块，页面显示〖待归档〗和〖已归档〗两个标签页。在〖待归档〗标签页中可以对已经流转结束的发文和收文进行归档操作；〖已归档〗页面可以查看已归档的发文和收文，在〖归档文件夹管理〗中，对归档文件夹有控制权限的用户，可以对已归档公文进行撤回操作，被撤回公文显示为未归档状态。

在待归档页面中，选择某一条公文，点击【归档】按钮，维护相关信息，打“*”为必填，维护完后点【归档】即完成归档。

进入〖已归档〗列表，点击操作栏中【修改文号】按钮，可以对已归档公文的文号进行修改。点击操作栏【撤回】按钮，可以将已归档的公文撤回，被撤回的公文显示在待归档列表中。

注意事项：

1.在人员权限设置中维护了归档人权限的用户才能看到待归档页面，并且要在归档文件夹中有文件夹控制权限才可以进行归档操作。

2.已归档公文修改文号后只在列表中更新显示文号信息，在详情中还是保持原来的。
