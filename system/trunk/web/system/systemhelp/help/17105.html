人员权限设置：

维护用户权限。

执行条件：

系统正常运行。

操作说明：

进入〖人员权限设置〗模块，页面显示用户权限设置列表，点击【设置】按钮，选择相应的用户，点保存即可。

部门/单位签收员：发送给部门/单位的公文，需要此权限才可以签收。

部门/单位登记员：发送给部门/单位的公文，需要此权限才可以登记。

归档人：有此权限才可以对公文进行登记，同时需要归档文件夹的控制权限，两者缺一不可。

单位流程管理员：有此权限可以在发文/收文流程管理中维护单位流程。

部门/单位发送员：有此权限可以在发文发送中查看到整个部门/单位所发送的公文。

印章管理员：有此权限可以维护印章。

公文发送修改员：有此权限可以在对结束流程的发文进行修改。

流转意见修改员：有此权限可以在对流转种的公文的流转意见进行修改。
