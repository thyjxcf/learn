收文转发：

对流转结束的收文进行转发。

执行条件：

系统正常运行，修改公文需具有公文发送修改员权限。

操作说明：

进入模块，页面显示〖未发送收文〗和〖已发送收文〗两个标签页。〖未发送收文〗页面显示流程已结束但是未发送的收文，〖已发送收文〗页面显示已发送的收文。

点击操作栏中【编辑】按钮，进入发送页面，点击【查看详情】可以查看公文详细信息。维护主送/抄送单位时，点击【+请选择】按钮，在弹出的通讯录界面中选择所需的主送/抄送用户，维护完后，点【发送】即可，当主送和抄送重复时，只保留主送发送。推送功能同发文一致，此处不再赘述。

有公文发送修改员权限的用户会显示【修改公文】按钮，点击【修改公文】，跳转到公文详情页面，修改公文基本信息和正文附件页面，修改完毕后，点击【保存】即可。

注意事项：

1.有公文发送修改员权限的用户会显示【修改公文】按钮，可以对公文进行修改，在〖人员权限设置〗页面进行赋权。

2.当主送和抄送重复时，只保留主送发送。
