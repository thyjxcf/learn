归档文件夹管理：

维护、查看发文和收文归档的文件夹。

执行条件：

系统正常运行。

操作说明：

进入〖归档文件夹管理〗模块,页面显示已维护的归档文件夹列表。点击【新增文件夹】按钮，维护文件夹相关信息，维护完后点保存即可。点击列表操作栏中【编辑】按钮，可以修改文件夹名称、访问权限和控制权限，点击【删除】按钮可删除该归档文件夹，归档文件夹中已有公文的，不可删除。

新增和修改归档文件夹时，点击访问权限和控制权限中的【选择】按钮，弹出〖通讯录〗页面，选择相关用户，点击【保存】，所选用户对当前归档文件夹拥有访问/控制权限，访问权限中，勾选所有人，即本单位所有用户对当前归档文件夹拥有访问权限。

注意事项：

1.归档人权限在人员权限设置模块中维护。

2.文件夹访问权限和控制权限，与人员权限设置中归档人权限有关：

归档人+访问权限=可以查询归档文件；

归档人+控制权限=可以对公文进行归档、撤回等操作；

非归档人+访问权限=可以查询归档文件；

非归档人+控制权限=不可查询，不可归档。
