收文流程管理：

维护单位和部门发文、收文流程步骤。

执行条件：

系统正常运行。

操作说明：

点击桌面上的【新建流程】按钮，维护流程名称和流程描述，点击【下一步，流程设计】按钮，进入流程设计页面。

如果当前流程是简单模式：鼠标移动到页面左侧【步骤】按钮，点击拖动步骤，将步骤拖动到相关范围内，会弹出新建步骤弹出框。维护步骤类型、步骤名称、步骤负责人等相关信息后，点击【确定】按钮，步骤就维护成功了。

如果当前流程是复杂模式：点击【添加步骤】，弹出新建步骤弹出框。维护步骤类型、步骤名称、步骤负责人等相关信息后，点击【确定】按钮，步骤就维护成功了。如果维护的步骤是并行步骤，需要用并行节点来并联成不同的分支，点击【添加并行节点】即可。

流程步骤维护结束后，点击页面右侧【确定】按钮，该流程保存成功；点击【返回】按钮或者页面右上角【关闭】按钮，该流程不保存直接返回流程列表。

在流程列表可以进行默认流程的维护，选择流程状态为成功的流程，直接在列表中勾选为默认流程即可。

注意事项：

1.人员权限设置中拥有单位流程管理员权限的用户可以维护和查看单位流程，普通用户只能查看和维护本部门流程，并且每个部门维护本部门相关的部门流程，不同部门之间的部门流程互不影响。

2.维护流程时，如果维护为并行步骤则需要维护多个负责人。

3.通过更改OFFICEDOC.FLOW.EASY.LEVEL参数可以切换流程模式。
