收文登记：

签收、登记来自同平台的公文，手工登记通过Email或其他方式收到的公文。

执行条件：

系统正常运行，有部门签收员/登记员权限的可以签收/登记发送到本部门的公文，有单位签收/登记员的可以签收/登记发送到本单位的公文。

操作说明：

在〖未登记〗列表中，选择要签收的公文，点击操作栏中【编辑】按钮，进行签收。收文签收后可进入到收文登记页面，维护相关信息后，点击【发送】按钮即可。如果当前单位维护了单位默认流程，可直接发送，否则要先点击【选择流程】按钮，选择收文流程；点击【保存草稿箱】按钮，该收文被保存为已签收状态，所维护相关收文登记信息被正确保存；点击【返回】按钮，直接返回未登记列表页，所维护相关收文登记信息不进行保存；点击【显示相关流程】按钮，弹出单位默认流程图。

点击列表页中的【新建手工登记】按钮，可以对外来收文进行手工登记，维护相关的登记信息，打“*”为必填，选择相关流程进入流转即可，选择流程进入流转方式同正常登记一致。

收文进入流转方式与发文一样。从登记页点击【选择流程】后，进入流程选择页面，默认显示单位流程中的默认流程，切换到部门流程则默认显示部门流程下的默认流程，用户可以根据自己的需要选择需要的流程。选择所需的收文流程后，点击【修改】按钮，进入流程修改页面，修改后，点击【确定流程】按钮，将已修改的流程保留并返回流程选择页面，点击【保存流程】按钮，将已修改的流程保存为一个新的流程到收文流程管理中，点击【返回】按钮，修改流程不保存直接返回流程选择页面。选择流程后，点击【确定】按钮，选中当前流程，返回公文编辑页面；点击【发送】按钮，收文流程启动成功，公文进入流转状态；点击【返回】按钮，不选中当前流程，直接返回到编辑页。

收文进入流转之后，可以在收文登记中〖已登记〗页面下进行查看。选择某一条公文记录，点击公文标题，进入公文详情页面，可以对该公文进行“催办”、“修改当前流程”、“设置代签人”、“设置代办人”、“查看流程图”、“查看处理单”和“返回列表”等操作。

注意事项：

1.收文登记需要在人员权限设置中维护用户角色，有部门签收员/登记员权限的可以签收/登记发送到本部门的公文，有单位签收/登记员的可以签收/登记发送啊都本单位的公文，有单位的权限不代表可以签收或登记部门的，发送给个人的可以直接签收。

2.通过发文发送到收文的公文在登记时，显示收文类型为阅办，手工登记的收文显示收文类型为督办。

3.本系统中发送的发文在登记时载正文默认都是pdf格式的，手工登记的收文下载正文有doc和pdf两种格式。

4.正常收发文的公文，不允许修改文号。

5.手工登记的收文可以维护和修改文号。

6.在列表中，未签收的记录显示为红色，已签收未登记的显示为黑色，紧急程度为“紧急”、“特急”和“特提”的，公文标题加粗显示。

7.ie浏览器下，如果不能编辑正文，请下载公文控件，并进行安装，安装成功后即可进行正文编辑。

8.非ie浏览器下，如果不能编辑正文，请下载公文控件和公文非ie控件，两个控件均要下载安装成功后才可进行正文编辑。

9.如安装了公文控件后还是不能正常启动控件，请下载控件清理注册工具一试。
