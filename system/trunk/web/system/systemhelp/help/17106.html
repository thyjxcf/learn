公文参数设置：

维护、查看公文文字代号、部门分类号、公文类型、模板和印章。

执行条件：

系统正常运行。

操作说明：

文字代号：进入模块，页面显示已维护的单位文号，点击第一行最后的【+】按钮，可新增一条记录，维护单位文号类型和排序号，点击【保存】即可。点击【—】按钮，可删除对应的记录，列表最少要有一条记录。在维护和修改过程中出现错误，并且未保存过的，点击【重置】可以还原到最初数据。

部门分类号：进入模块，显示单位代号和各部门代号的维护页面，维护单位和各个部门的代号，点击【保存】即可。在维护和修改过程中出现错误，并且未保存过的，点击【重置】可以还原到最初数据。

公文类型设置：进入模块，页面显示公文类型列表，列表中初始公文类型由微代码直接生成，点击维护栏中【可维护】按钮，也可修改公文类型，维护完后点【保存】即可。点击页面左下角【新增】按钮，可以新增公文类型，维护相关信息，维护完后点【保存】即可。

模板管理：进入模块，页面显示套红模板列表。模板分系统、单位、部门和个人四种，系统模板为系统默认设置，不可增删改。选择非系统类型的模板，可以进行新增操作，点击【新建模板】，维护相关信息，维护完后点【保存】即可，也可以点击【模板导入】，导入已维护好的套红模板。点击列表中【编辑模板】按钮，可以对模板进行修改，点击列表中【预览模板】，可以预览。

印章管理：添加公文流转所需要的电子印章。进入模块，点击【新增印章】，维护相关信息，维护完后点【保存】即可。所创建的印章，创建者默认显示在使用人中，并可以选择其他的使用者，在“用印”时要使用某一印章，将用户添加到使用人中即可。列表显示已维护的印章记录。点击操作栏中【编辑】按钮，可以修改印章相关信息，点击【修改个人使用者密码】可将原密码修改为新密码，点击【清空使用人密码】，将密码清空，修改完后点保存即可。

注意事项：

1.套红模板预览时，如果当前用户安装的办公软件是wps，预览失败的可能性比较大，openoffice对大多数的wps文件都无法进行转换。

2.新一个套红建模板文件，必须要包含正文书签，否则无法保存成功。

3.不同单位、部门、个人的套红模板互相之间不可见，即假设a用户维护的个人模板，b用户不可见。

4.单位类型的模板，本单位有模块权限的用户均可增删改，本单位所有用户都可以使用该类型的模板；部门类型的模板，本部门有模块权限用户均可以增删改，本部门所有用户都可以使用该类型的模板；个人类型的模板只能本人增删改，也只能本人使用。

5.如果是初次使用本系统，在新增印章时系统会提示加载控件，请先点击“下载”安装印章管理控制后，刷新页面即可，目前只支持ie浏览器。

6.印章图片可以使用专门的印章制作软件进行制作，且图片必须为.gif或.jpg格式。
