发文发送：

对流转结束的公文进行发送。

执行条件：

系统正常运行。

操作说明：

进入模块，页面显示〖未发送发文〗和〖已发送发文〗两个标签页。〖未发送〗页面列表显示流程已结束但是未发送的发文，〖已发送〗列表显示已发送的发文。

点击操作栏中【发送】按钮，进入发送页面，维护主送/抄送单位时，点击【+请选择】按钮，弹出相关通讯录界面，选择所需的主送/抄送用户，维护完后，点击【发送】即可。

有公文发送修改员权限的用户会显示【修改公文】按钮，点击【修改公文】按钮，可跳转到修改公文基本信息和正文附件页面。点击〖已发送发文〗标签，进入页面，可以对已发送的公文进行 “催办”、 “撤回”等操作。“催办”时，系统会自动向签收人发送催办通知；“撤回”后，会将已发送的发文撤回，发文接收人处该公文记录依然存在，操作栏显示为“已撤回”状态，不能对已撤回的公文做任何操作

注意事项：

1.当主送和抄送重复时，只保留主送发送，不会出现重复发送的情况。

2.发文流水号是根据年份+序号方式生成。
