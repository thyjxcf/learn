收文管理：

对本单位所有收文进行管理，可以进行查看、删除等操作。

执行条件：

系统正常运行。

操作说明：

进入模块，〖全部〗标签页列表中显示本单位所有收文；〖流转中〗标签列表中显示正在流转的收文；〖未归档〗和〖已归档〗标签下又分为未发送和已发送两类，对流转结束的公文是否发送和归档进行分类。

〖收文管理〗中所有收文都可以进行查看、删除等操作，勾选要删除的收文，点击页面左下方【删除】按钮，所选收文进入收文回收站中。流转状态为待归档的收文，操作栏显示【拟办意见】、【批办意见】和【已办科室】等按钮；流转状态为流转中的收文，操作栏显示【拟办意见】、【批办意见】、【已办科室】、【未办科室】、【修改正文】和【催办】等按钮。

对流转中状态的收文进行修改操作，点击操作栏中【修改公文】按钮，进入详情页面，修改公文基本信息和正文附件等信息，修改完毕后点【保存公文】即可。如果用户有流程意见修改权限的，则操作列表中的“拟办意见”、“批办”意见等合并为“查看意见”，点击操作栏中的【查看意见】按钮，可以查看各流程意见并进行修改

在〖流转中〗页面，点击公文标题进入详情页面，可以对公文当前流程进行修改，如果下一步骤流程为并行步骤，则弹出提示框选择其中一个步骤，点击【修改流程】进入流程修改页面，修改完毕后点击【确定】即可。

注意事项：

1.如果用户有流程意见修改权限的，则操作列表中的“拟办意见”、“批办”意见等合并为“查看意见”。
