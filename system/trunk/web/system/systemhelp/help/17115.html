发文管理：

对本单位所有发文进行管理，可以进行查看、删除等操作。

执行条件：

系统正常运行。

操作说明：

进入模块，页面显〖全部〗、〖已流转〗、〖未归档〗和〖已归档〗四个标签页，〖全部〗标签页列表中显示本单位所有发文，〖流转中〗标签列表中显示正在流转的发文，〖未归档〗和〖已归档〗标签下又分为未发送和已发送两类，对流转结束的公文是否发送和归档进行分类。

〖发文管理〗中点击公文标题可以查看公文详情，选择公文点击页面左下方【删除】按钮，所选发文进入发文回收站中。点击操作栏中【修改公文】按钮，可以对流转中状态的公文进行修改。如果用户有流程意见修改权限，点击办理情况中流程步骤，在弹出意见框中点击操作栏中【修改】按钮可以对流程意见进行修改。

在〖流转中〗页面，选择需要公文，点击左下方的【催办】按钮可以对公文进行催办，点击公文标题进入详情页面，点击【修改流程】按钮可以对公文当前流程进行修改。

注意事项：

1.发文管理中如果点击正文下载标识无反应，表示原先是有正文的但是目前正文附件可能已经丢失，如果没有正文的公文会直接显示无正文。

2.在发文管理中可以删除各个状态下的公文，被删除的公文在发文回收站中可以看到，并且被删除的公文可以还原，原来是什么状态还原后还是什么状态。

3.人员权限设置中部门发送员指有该权限的用户可以看到部门发送的公文，单位发送员指有该权限的用户可以看到本单位发送的公文。
