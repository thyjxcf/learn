1、院系树：单位 -> 院、系
2、部门树：单位 -> 院、系 -> 部门、教研组 -> 教师
3、专业树：单位 -> 院、系 -> 专业
4、方向树：单位 -> 院、系 -> 专业 -> 专业方向
5、年级树：单位 -> 院、系 -> 专业 -> 专业方向 -> 年级
6、班级树：单位 -> 院、系 -> 专业 -> 专业方向 -> 年级 -> 班级
7、学生树：单位 -> 院、系 -> 专业 -> 专业方向 -> 年级 -> 班级 -> 学生
8、教学区树：单位 -> 教学区
9、教学场地树：单位 -> 教学区 -> 教学场地
10、教学资源树：单位 -> 教学区 -> 教学场地 -> 教学资源
